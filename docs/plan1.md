Система мониторинга коммерческой недвижимости

## Суть проекта

Сервис автоматически находит объекты коммерческой недвижимости (офисы, склады, торговые помещения) по цене значительно ниже рыночной — от 20% дешевле. Риелтор получает уведомление в течение нескольких минут после появления такого объекта.

## Ключевые функции

### 1. Парсер
- Источники данных: CIAN + Avito (настраиваемый список городов)
- Собирает объекты, появившиеся или отредактированные за последний час
- Данные: цена, площадь, адрес, тип объекта, фото, контакты продавца
- Геокодирование через Яндекс.Карты API
- Защита от блокировок: ротация прокси, рандомные задержки

### 2. Анализатор
- Для каждого объекта находит N похожих объектов того же типа в радиусе X км
- Вычисляет "рыночную цену" как медиану стоимости кв.м. соседей
- Определяет отклонение вниз (на сколько % объект дешевле рынка)
- Хранение истории: 6 месяцев

### 3. Алертер (мгновенные уведомления)
- Отправляет карточки объектов в Telegram
- Настраиваемый порог срабатывания (по умолчанию 20%)
- Настраиваемое рабочее время (например 9:00-21:00 МСК)
- Группировка объектов в пачку (если найдено несколько за час)

### 4. Дайджестер (утренняя рассылка)
- Отправляет на email в 9:00 утра (МСК)
- Только объекты, недооценённые более чем на порог
- Все найденные за ночь

### 5. Веб-интерфейс
- Список всех объектов с фильтрами
- Статусы объектов: Новый / В работе / Просмотрено / Отклонено
- Комментарии к объектам
- Настройка городов, порогов, временных окон (глобально)

## Алгоритм работы

Парсер (каждый час):
  → Новые/изменённые объекты → SQLite

Анализатор (каждые 30 мин):
  → Для каждого нового объекта:
     → Найти N похожих в радиусе X км того же типа
     → Вычислить медиану цены кв.м.
     → Если отклонение > порог → пометить как "недооценён"

Алертер (в рабочее время):
  → Группа недооценённых объектов за последний час
  → Отправить карточки в Telegram

Дайджестер (9:00 утра):
  → Все недооценённые объекты за ночь
  → Отправить на email

## Формат карточки объекта

- Фото объекта
- Цена (руб.) + цена за кв.м.
- Адрес (геокодированный)
- Площадь (кв.м.)
- Тип объекта
- Отклонение от рынка (%)
- Ссылка на источник
- Контакты продавца

## Статусы объектов

- Новый — только что найден
- В работе — взяли в работу
- Просмотрено — осмотрели / связались
- Отклонено — не подошло

## Технические параметры (настраиваемые)

- Порог отклонения: 20% (по умолчанию)
- Радиус поиска соседей: X км (настраивается)
- Количество соседей для анализа: N (настраивается)
- Рабочее время: настраивается (по умолчанию 9:00-21:00 МСК)
- Время утреннего дайджеста: 9:00 (МСК)
- Срок хранения истории: 6 месяцев
